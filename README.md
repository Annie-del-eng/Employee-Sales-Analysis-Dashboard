
# Employee Sales Performance Analysis

This project is an interactive sales performance dashboard developed using Microsoft Power BI. It analyzes a synthetic dataset of 100 employees to provide insights into sales, profits, target achievement, product categories, regional performance, and customer feedback. The dashboard includes KPI cards, charts, maps, and slicers for easy data exploration and analysis. It helps identify top-performing employees, compare business performance across regions and products, and support informed decision-making through interactive visualizations.

**Tools Used:** Microsoft Power BI, Microsoft Excel

**Dataset:** Synthetic employee sales dataset (100 records)

**Purpose:** To demonstrate the use of Business Intelligence and data visualization for sales performance analysis.
